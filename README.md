# CodeAlpha E-Commerce Store

## Description
A simple E-Commerce Store built using HTML, CSS, JavaScript, Node.js and Express.js.

## Features
- User Login
- Product Listing
- Product Details
- Shopping Cart
- Checkout
- Order Processing

## Technologies
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## How to Run

```bash
npm install
npm start
```

Open in browser:

```
http://localhost:3000
```
